Histology application (first attempt)


My first attempt using the library OpenSeaDragon with HTML5 to automatically change the position of a histology slide based on timestamps in the video.

The current application does not use working histology slides, however use of the deepzoomAPI has been attempted in generating the imaginary slides. Further work involves the addition of a scalebar and the use of JSON objects in listing time/slide locations.
